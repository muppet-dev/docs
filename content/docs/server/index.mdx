Fast, lightweight, built on Web Standards. Support for any JavaScript runtime.

```ts
import { Muppet } from "muppet";

const mcp = new Muppet({
  name: "my-mcp-server",
  version: "1.0.0",
});

mcp.connect(new StdioServerTransport());
```
